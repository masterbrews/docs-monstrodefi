# mUSDC

mUSDC is a non-upgradeable ERC-4626 vault that accepts USDC deposits and issues mUSDC shares. The vault is designed so backing per share can increase over time as fees and integrated protocol revenue enter the vault.

mUSDC starts inside the Monstro ecosystem, with Based Loans as its first major integration, but is intended to be useful across external products, partner integrations, and liquidity venues.
