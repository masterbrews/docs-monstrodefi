# xMONSTRO

xMONSTRO is a non-upgradeable, fully permissionless bonding-curve vault token backed by MONSTRO. Every mint burns MONSTRO and adds backing to a shared vault, so the MONSTRO behind each xMONSTRO is designed to grow over time.

xMONSTRO is a whole-unit token with no fractional units. It has no owner, no admin, and no pause switch. All parameters are immutable constants set at deployment.
