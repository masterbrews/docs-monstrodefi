# Staking

## Staking $MONSTRO

The $MONSTRO staking system is a core component of the Monstro ecosystem.\
It is fully on-chain, non-custodial, and designed to reward long-term participation while remaining transparent and flexible.

Staking allows $MONSTRO holders to earn emissions, unlock tier-based bonuses, and align with the long-term growth of the protocol without surrendering custody of their tokens.

This section serves as a central hub for all staking-related actions and information.

***
